# API Server

Backend API for data coordination between IoT devices and dashboard

## Overview

Responsible for receiving data from devices, managing databases, and delivering data to the dashboard.

## Key Features

- Receiving and storing device data
- RESTful API endpoints
- WebSocket server
- Authentication and authorization
- Data aggregation and analysis

## Tech Stack

- Runtime: Node.js/Python/Go, etc.
- Framework: Express/FastAPI/Gin, etc.
- Database: PostgreSQL/MongoDB/TimescaleDB, etc.

## Setup

```bash
cd apps/api
npm install
npm run dev
```
